## Input

<p>The input consists of a single line, which contains two integers $n$, $x$ ($2 \leq n \leq 4000$, $1 \leq x &lt; 1\,000\,000\,007$).</p>

## Output

<p>Let $a_{m,k}$ be the answer for the pair $(m, k)$, modulo $998\,244\,353$.</p><p>Let $$\large S = \sum_{m=3}^{n+1} \sum_{k=0}^{n-1} a_{m,k}x^{mn+k}\phantom{0}.$$</p><p>Output a single line with an integer: $S$ modulo $1\,000\,000\,007$.</p><p>Note that using two different modulos is intentional. We want you to calculate all the $a_{m,k}$ modulo $998\,244\,353$, then treat them like integers in the range $[0, 998\,244\,352]$, and hash them modulo $1\,000\,000\,007$.</p>
