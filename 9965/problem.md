## Description

<div><p>In a well-known school, a physical education lesson took place. As usual, everyone was lined up and asked to settle in "the first–$k$-th" position.</p><p>As is known, settling in "the first–$k$-th" position occurs as follows: the first $k$ people have numbers $1, 2, 3, \ldots, k$, the next $k - 2$ people have numbers $k - 1, k - 2, \ldots, 2$, the next $k$ people have numbers $1, 2, 3, \ldots, k$, and so on. Thus, the settling repeats every $2k - 2$ positions. Examples of settling are given in the "Note" section.</p><p>The boy Vasya constantly forgets everything. For example, he forgot the number $k$ described above. But he remembers the position he occupied in the line, as well as the number he received during the settling. Help Vasya understand how many natural numbers $k$ fit under the given constraints.</p><p>Note that the settling exists if and only if $k &gt; 1$. In particular, this means that the settling <span class="tex-font-style-bf">does not exist</span> for $k = 1$.</p></div><div class="input-specification"><p>Each test consists of multiple test cases. The first line contains a single integer $t$ ($1 \leq t \leq 100$) — the number of test cases. This is followed by the description of the test cases.</p><p>The only line of each test case contains two integers $n$ and $x$ ($1 \le x &lt; n \le 10^9$) — Vasya's position in the line and the number Vasya received during the settling.</p></div><div class="output-specification"><p>For each test case, output a single integer — the number of different $k$ that fit under the given constraints.</p><p>It can be proven that under the given constraints, the answer is finite.</p></div>

## Input

<p>Each test consists of multiple test cases. The first line contains a single integer $t$ ($1 \leq t \leq 100$) — the number of test cases. This is followed by the description of the test cases.</p><p>The only line of each test case contains two integers $n$ and $x$ ($1 \le x &lt; n \le 10^9$) — Vasya's position in the line and the number Vasya received during the settling.</p>

## Output

<p>For each test case, output a single integer — the number of different $k$ that fit under the given constraints.</p><p>It can be proven that under the given constraints, the answer is finite.</p>





```input1|2,4,6
5
10 2
3 1
76 4
100 99
1000000000 500000000
```




```output1
4
1
9
0
1
```
